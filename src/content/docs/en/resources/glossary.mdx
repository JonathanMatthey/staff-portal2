# Glossary

A comprehensive glossary of terms related to Interledger Protocol, blockchain, and the Interledger Foundation.

## Interledger Terms

### Interledger Protocol (ILP)
An open protocol suite for sending payments across different ledgers and payment networks.

### Connector
A node in the Interledger network that routes payments between different ledgers.

### Payment Pointer
A standardized identifier for payment accounts, similar to an email address for payments.

### STREAM Protocol
A transport protocol for Interledger that provides for sending multiple packets of money and data over ILP.

### Web Monetization
A proposed API standard that allows websites to request small payments from users automatically.

## Payment & Blockchain Terms

### Ledger
A record-keeping system for financial transactions.

### Settlement
The completion of a transaction where assets are transferred from one party to another.

### Liquidity
The availability of liquid assets or cash that can be used for transactions.

### Atomic Transaction
A transaction that either completes entirely or not at all.

## Technical Terms

### API (Application Programming Interface)
A set of protocols and tools for building software applications.

### Open Source
Software with source code that anyone can inspect, modify, and enhance.

### SDK (Software Development Kit)
A collection of software development tools in one installable package.

### Webhook
A method for applications to provide real-time information to other applications.

## Foundation Terms

### Grant Program
Funding opportunities provided by the foundation to support ecosystem development.

### Community Grant
Funding provided to community members for projects that benefit the Interledger ecosystem.

### Ambassador Program
A program that empowers community members to represent and promote Interledger.
