### Icon Only
The Interledger icon without text, for small applications like favicons and social media avatars.

## File Formats

Logos are available in multiple formats:

- **SVG**: Scalable vector format, best for web and digital use
- **PNG**: Raster format with transparency, for general use
- **JPG**: Raster format without transparency
- **EPS**: Vector format for print and professional design work

## Usage Guidelines

### Do's
- ✓ Use official logo files
- ✓ Maintain adequate clear space around the logo
- ✓ Use on appropriate backgrounds
- ✓ Scale proportionally

### Don'ts
- ✗ Don't distort or stretch the logo
- ✗ Don't change the colors
- ✗ Don't add effects or shadows
- ✗ Don't place on busy backgrounds that reduce visibility

## Clear Space

Maintain a minimum clear space around the logo equal to the height of the "I" in "Interledger."

## Minimum Size

### Digital
- Minimum width: 120px

### Print
- Minimum width: 1 inch (2.54cm)
